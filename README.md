# StellarMap
StellarMap is an open source, cross-platform PyQt desktop app used to explore the Stellar blockchain network.
The app makes API calls via **horizon.stellar.org** and **api.stellar.expert**

<img src="/docs/source/images/stellarmap_prototype_v0_1_2.PNG" width="100%" title="StellarMap">

## What can it do?

 - identify upstream creator accounts given a Stellar account
 <img src="/docs/source/images/stellarmap_prototype_data_v0_2_0_doge-token_com.PNG" width="100%" title="StellarMap DATA tab">
 - display json outputs of current session
 <img src="/docs/source/images/stellarmap_prototype_json_v0_2_0_doge-token_com.PNG" width="100%" title="StellarMap JSON tab">
 - display terminal outputs of current session
 <img src="/docs/source/images/stellarmap_prototype_terminal_v0_2_0_doge-token_com.PNG" width="100%" title="StellarMap TERMINAL tab">
